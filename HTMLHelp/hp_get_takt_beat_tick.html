HP_GetTaktBeatTick

Computes takt, beat and tick by a given time-stamp.

HP_MIDIFILE_API UINT HP_GetTaktBeatTick
(
HP_MIDIFILE_MODULE *hphandle,
int time,
int * takt,
int * beat,
int * tick
);

HP_MIDIFILE_API UINT HP_GetTaktBeatTick
(
HP_MIDIFILE_MODULE *hphandle,
int time,
int time_of_last_time_signature,
int takt_of_last_time_signature,
int num,
int denum,
int * takt,
int * beat,
int * tick
);

Parameters

time The time-stamp, i.e. the number of ticks from the beginning.
takt A pointer to the takt, filled by the function.
beat A pointer to the beat, filled by the function..
tick A pointer to the ticks since last beat, filled by the function.
time_of_last_time_signature The time-stamp of the last time-signature event.
takt_of_last_time_signature The takt of the last time-signature event.
num The value of num, got with the the last time-signature event.
denum The value of num, got with the the last time-signature event.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

The second implementation of HP_GetTaktBeatTick is a efficient method to be applied when scanning the eventlist.
