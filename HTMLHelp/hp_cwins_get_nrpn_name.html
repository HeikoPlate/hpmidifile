HP_CWInsGetNRPNName

To get the name of NRPN.

HP_MIDIFILE_API UINT HP_CWInsGetNRPNName
(
HP_MIDIFILE_MODULE *hphandle,
int nrpn,
char **nrpn_name
);

Parameters


hphandle Pointer to a HP_MIDIFILE handle.
rpn The number of the NRPN, which is build by the formula "MSB*128+LSB".
MSB and LSB are the values of the Controller Events no. 99 and no. 98.
nrpn_name The name of the NRPN, filled by the function.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.
