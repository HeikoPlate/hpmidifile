Glossary

delta-time The number of ticks since the previous time-stamp.
MIDI-event Elementary sequence of bytes with a specific function to be interpreted
by a MIDI sound-generator. In a MIDI-file the events are preceded by the
delta-time, related to the previous event. delta-time specifies the time
when the event must be sent.
MIDI-file session All function-calls connected with only one handle.
A session is started with the call of HP_INIT() to get this handle and
ends when deleting the handle with the call of HP_FREE(&handle).
SMF0 Standard MIDI-file of format 0.
SMF1 Standard MIDI-file of format 1.
time-stamp The number of ticks of an MIDI-event since start.
track The sequence of MIDI-Events of a MTrk-chunk. The time is specified
independant from other tracks. MIDI-files of format 0 (SMF0) have
exact one track, MIDI-files of format 1 (SMF1) use more than one track.
