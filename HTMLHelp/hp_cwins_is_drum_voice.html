[all functions]

HP_CWInsIsDrumVoice

To find out if a voice is a drumkit.

HP_MIDIFILE_API UINT HP_CWInsIsDrumVoice
(
HP_MIDIFILE_MODULE *hphandle,
long bank_no,
int voice_no,
int* is_drum
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
bank_no The bank number of the voice.
voice_no The voice number of the voice.
is_drum Filled by the function: drumkit 1, melody voice 0

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.
