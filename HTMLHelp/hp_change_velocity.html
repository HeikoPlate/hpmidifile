HP_ChangeVelocity

To change velocity of a note event, identified by its index "id" in the eventlist.

HP_MIDIFILE_API UINT HP_ChangeVelocity
(
HP_MIDIFILE_MODULE *hphandle,
int id,
int chan,
int note,
int value,
bool percent
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
id id >= 0: Change for this note-event's index only.
id = HP_ALL_EVENTS: Change for all note events.
chan chan >= 0: Change for this MIDI-channel,
chan = HP_ALL_CHAN: Change for all MIDI-channels.
note note >= 0: Change for this note,
note = HP_ALL_NOTES: Change for all note-values.
value Specificies the value for the new velocity. Look at percent.
percent percent = HP_PERCENT: Change old velocity with percent-value, 150: +50%, 60: 60%
percent = HP_NO_PERCENT: Change old velocity by the new value 0 < value <= 127.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

Use HP_ChangeNote to change the note-value.
