[all functions]

HP_Select

To select one or all MIDI-events.

HP_MIDIFILE_API UINT HP_Select
(
HP_MIDIFILE_MODULE *hphandle,
int id = HP_ALL_EVENTS
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
id Index of the MIDI-event. id = HP_ALL_EVENTS: All MIDI-events will be selected.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

The index id is the position of the MIDI-Event in the eventlist of HP MIDIFILE. We get this id with the function HP_ReadEvent of the group "Stream".
