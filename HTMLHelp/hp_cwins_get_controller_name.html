HP_CWInsGetControllerName

To get the name of a Controller Event.

HP_MIDIFILE_API UINT HP_CWInsGetControllerName
(
HP_MIDIFILE_MODULE *hphandle,
int controller,
char** controller_name
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
controller The number of a Controller Event.
controller_name The name of the Controller Event, filled by the function.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

The function allocates memory for controller_name. The memory has to be destroyed by HP_Delete() afterwards.
