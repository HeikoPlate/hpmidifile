HP_InsertXFVersionID

To insert a XF Version ID event.

HP_MIDIFILE_API UINT HP_InsertXFVersionID
(
HP_MIDIFILE_MODULE *hphandle,
int time,
unsigned char *event_data,
int length
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
time The time (number of ticks) of the position of time of the XF Version ID event.
event_data An array with the bytes of the event.
length The number of bytes.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

The event data must be correct specified, beginning with 0xff and ending with the last byte of the event.
