[all functions]

HP_ChangePPQN

To set or change PPQN with a new value.

HP_MIDIFILE_API UINT HP_ChangePPQN
(
HP_MIDIFILE_MODULE *hphandle,
int ppqn
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
ppqn The new value. Use the defines HP_PPQN48, HP_PPQN72, HP_PPQN96, HP_PPQN120, HP_PPQN144, HP_PPQN168, HP_PPQN192, HP_PPQN216, HP_PPQN240, HP_PPQN360, HP_PPQN384, HP_PPQN480, HP_PPQN960, HP_PPQN1920 only. .

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.
