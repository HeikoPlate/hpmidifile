HP_ReadChannelPressure

To read all attributes of a Channel Pressure event, identified by its index "id" in the eventlist.

HP_MIDIFILE_API UINT HP_ReadChannelPressure
(
HP_MIDIFILE_MODULE *hphandle,
int id,
int *time,
int *chan,
int *pressure
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
id The index of the eventlist of the Channel Pressure event.
time A pointer filled by the function with the event's position of time.
chan A pointer filled by the function with the number of the MIDI-channel.
pressure A pointer filled by the function with the pressure.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.
