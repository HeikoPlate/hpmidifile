HP_GetTimeFromTaktBeatTick

Computes the time-stamp time by given takt, beat and tick.

HP_MIDIFILE_API UINT HP_GetTimeFromTaktBeatTick
(
HP_MIDIFILE_MODULE *hphandle,
int takt,
int beat,
int tick,
int * time
);

Parameters

takt Takt number.
beat Beat number.
tick Number of ticks since the last beat.
time A pointer to the variable time, filled by the function..

Return Value

Error code. On success the code HP_ERR_NONE.
