HP_CWInsGetBankVoiceNoteName

To get the names of the voicebank, patch, note, drum-kit and drum.

HP_MIDIFILE_API UINT HP_CWInsGetBankVoiceNoteName
(
HP_MIDIFILE_MODULE *hphandle,
int bank,
int voice,
int note,
char **bank_name,
char **voice_name,
char ** note_name
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
bank The number of the voice-bank, which is build by the formula "MSB*128+LSB".
MSB and LSB are the values of the Controller Events no. 0 and no. 32.
To get the GM voices, use bank = HP_GM_VOICE_BANK; for GM drums use bank = HP_GM_PERC_BANK.
voice The voice-number. For GM drums use voice = 0.
note The note- or drum-number.
bank_name The name of the voice-bank. Filled by the function.
voice_name The name of the voice or drum-kit. Filled by the function.
note_name The name of the note or drum. Filled by the function.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

If the function succeeds, it allocates memory for bank-name, voice-name and note-name, which must be destroyed by HP_Delete(..) for each name.
