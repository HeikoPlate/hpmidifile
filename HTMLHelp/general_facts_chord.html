Chord Event: General Facts



The parameters of Chord events are defined with the four bytes chord root cr, chord type ct, on-bass note bn and bass chord bt. Inserting and changing Chord events with HP MIDIFILE, these bytes have to be specified. Reading a Chord event, we get back these bytes and also a string with the text interpretation of the chord. The function HP_ConvertChordToText converts the four bytes to the chord-name.
