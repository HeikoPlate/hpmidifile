HP_ChangeAftertouch

To change the MIDI-channel and note-value of a note event, identified by its index "id" in the eventlist.

HP_MIDIFILE_API UINT HP_ChangeAftertouch
(
HP_MIDIFILE_MODULE *hphandle,
int id,
int note,
int value,
bool percent
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
id id >= 0: The index of the aftertouch event to be changed.
id = HP_ALL_EVENTS: Change all aftertouch events
note 0 <= note <= 127: New value of note for the aftertouch event.
note = HP_NO_NOTE: Do not change the note value for the aftertouch event
value Specification of the value used with change, look at percent
percent Specification how to change the pressure
percent = HP_PERCENT: Change old pressure with percent-value, 150: +50%, 60: 60%
percent = HP_NO_PERCENT: Change old pressure by the new value 0 < value <= 127.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.
