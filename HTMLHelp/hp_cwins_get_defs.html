HP_CWInsGetDefs

To get a list of all names of MIDI-instruments, supported with the loaded Instrument Definition File.

HP_MIDIFILE_API UINT HP_CWInsGetDefs
(
HP_MIDIFILE_MODULE *hphandle,
HP_CWDEF ** defs,
int no_defs
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
defs A pointer to an array of HP_CWDEF structures, which is filled by the function.
no_defs The number of instruments; i.e. the length of the array. Filled by the function.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

The function allocates memory for the array "defs". The user has to destroy the data afterwards with "HP_Delete(defs);".
