HP_ReadInstrument

To read the text of a Instrument event, identified by its index "id" in the eventlist.

HP_MIDIFILE_API UINT HP_ReadInstrument
(
HP_MIDIFILE_MODULE *hphandle,
int id,
int *time,
char **text
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
id The index of the eventlist of the Instrument event.
time A pointer filled by the function with the event's position of time.
text A pointer of a string filled by the function with the text of the event.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

The function allocates memory for the string. It has to be destroyed afterwards by the user with "HP_Delete(text);".
