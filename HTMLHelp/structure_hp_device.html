Structure: HP_DEVICE

To get an available MIDI-device with its ID and name.

typedef HP_device struct
{
int device_id;
char device_name[32];
} HP_DEVICE;

Selectors

device_id The ID of a MIDI-device.
device_name The name of the MIDI-device.
