[all functions]

HP_SelectMIDIDevice

To select a MIDI-device for play MIDI events.

HP_MIDIFILE_API UINT HP_SelectMIDIDevice
(
int id
);

Parameters

id The ID of a MIDI output device.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

The selection is global for all MIDI sessions.
The inital MIDI-device of HP MIDIFile is the Microsoft MIDI-Mapper.
