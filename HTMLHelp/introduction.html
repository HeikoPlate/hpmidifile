Introduction



You have downloaded HP MIDIFILE.

A dynamic link library to program applications for MIDI-files of types SMF0 or SMF1.

http://www.heikoplate.de/hpm

The library HP MIDIFILE allows to load existing MIDI-files or create new ones, to read, insert, delete, change all MIDI-events of GM1/GM2 and of YAMAHA XG/XF. With functions Select, Copy and Paste sets of events can be inserted at other time-points into the same or another MIDI-file. There are functions to find and select MIDI-devices and to play the loaded song.

HP MIDIFILE is intended to simplify the creation and editing of MIDI-files. Most of the functions work with the logic of events, not with internal data.

HP MIDIFILE offers functions to use Instrument Definition Files of the sequencer-software Cakewalk/Sonar.

The user of this library must be familiar with the structure of MIDI and MIDI-files. Applying YAMAHA SysEx and meta-events he should look at XG- and XF-Specifications. A lot of good documents can be found in the web.

HP MIDIFILE is implemented for the programming language Visual C++ for Windows.
