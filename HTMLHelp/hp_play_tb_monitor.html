HP_PlayTBMonitor

To listen at takt- and beat- events during the play of an MIDI-file. The function must be realized by the user.

void HP_PlayTBMonitor
(
int takt,
int beat
);

Parameters

takt The actual takt-number, starting with 0.
takt = -1: The play is finished.
beat The actual beat-number, starting with 0.
beat = -1: The play is finished.

Return Value
