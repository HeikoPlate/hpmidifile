HP_ChangeNote

To change the MIDI-channel and note-value of a note event, identified by its index "id" in the eventlist.

HP_MIDIFILE_API UINT HP_ChangeNote
(
HP_MIDIFILE_MODULE *hphandle,
int id,
int chan,
int note,
bool absolute
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
id id >= 0: The index of the eventlist of the note event.
id = HP_ALL_EVENTS: Change all note events of selected MIDI-channel chan.
chan The MIDI-channel of the note event(s) to be changed.
note Specification how to change the note value
absolute = HP_ABSOLUTE: new note value. note >= 0 provided.
absolute = HP_RELATIVE: increment (note > 0) or decrement (note < 0) by note steps.
absolute Look at parameter note

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

Use HP_ChangeNoteLength,to change the length.
Use HP_ChangeVelocity, to change the velocity.
