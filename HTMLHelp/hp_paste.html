HP_Paste

To paste all MIDI-events of the global buffer into a MIDIFile-session. We can specifiy, if the events of one or all MIDI-channels will be deleted in the target-region before.

HP_MIDIFILE_API UINT HP_Paste
(
HP_MIDIFILE_MODULE *hphandle,
int time,
bool delete_in_dest,
int del_events_chan
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
time Starting time in the destination for the events to be inserted.
delete_in_dest HP_DELETE: The events in the copy-region will be deleted before.
HP_NO_DELETE: No delete of events in the copy-region before.
del_events_chan Only if delete_in_dest == HP_DELETE.
del_events_chan==HP_NO_CHAN: All events not associated with a channel are deleted.
del_events_chan==HP_ALL_EVENTS: All events are deleted.
0 <= del_events_chan <= 15: All events associated with the channel are deleted.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

If the PPQN of the MIDI-events in the global buffer differs from the PPQN in the target, the PPQN of the target will be changed to the buffer PPQN.
