HP_Init

The call of this function starts a MIDI-file session. The returned pointer is the first parameter for most function-calls of the same session.

HP_MIDIFILE_API UINT HP_Init
(
);

Parameters

Return Value

On success a pointer to a handle of type HP_MIDIFILE_MODULE. NULL for a bad call.

Remarks

It is allowed to have more than one parallel MIDI-File session.
