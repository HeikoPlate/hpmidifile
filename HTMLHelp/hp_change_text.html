HP_ChangeText

To change the text of a Text event, identified by it's index of the eventlist.

HP_MIDIFILE_API UINT HP_ChangeText
(
HP_MIDIFILE_MODULE *hphandle,
int id,
char *new_text
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
id The index of the eventlist of the Text event.
new_text The new text.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.
