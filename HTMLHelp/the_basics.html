Downloaded files and how to install these



The package HP MIDIFILE consists of six files

  1. The header: HP_midifile.h
  2. The Microsoft-library: HP_midifile.lib
  3. The COFF-library: HP_midifile.exp
  4. The dynamic link library: HP_midifile.dll
  5. The map file HP_midifile.map with the decorated names
  6. The documentation file you are reading: hpmidifile.chm
What to do?

  1. Add the header HP_midifile.h to the files of your project and insert #include "HP_midifile.h" into your code.
  2. Copy the library HP_midifile.lib (or HP_midifile.exp ) into the library-path and link it with your VC++ project.
  3. Copy the DLL HP_midifile.dll into the folder of your executable file.

 

Important hint!

The application-software must not use the symbol HP_MIDIFILE_EXPORTS.
