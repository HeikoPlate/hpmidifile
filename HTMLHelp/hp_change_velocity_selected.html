HP_ChangeVelocitySelected

To change the velocity of selected note events.

HP_MIDIFILE_API UINT HP_ChangeVelocitySelected
(
HP_MIDIFILE_MODULE *hphandle,
int value,
bool percent
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
value Specificies the value for the new velocity. Look at percent.
percent percent = HP_PERCENT: Change old velocity with percent-value, 150: +50%, 60: 60%
percent = HP_NO_PERCENT: Change old velocity by the new value 0 < value <= 127.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

Use HP_ChangeNote to change the note-value.
