HP_ReadTempo

To read the attributes of a Tempo event, identified by its index "id" in the eventlist.

HP_MIDIFILE_API UINT HP_ReadTempo
(
HP_MIDIFILE_MODULE *hphandle,
int id,
int *time,
int *bpm,
int *cps
);

HP_MIDIFILE_API UINT HP_ReadTempo
(
HP_MIDIFILE_MODULE *hphandle,
int id,
int *time,
double *bpm,
int *cps
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
id The index of the eventlist of the Tempo event.
time A pointer filled by the function with the time of the event.
bpm A pointer filled by the function with the number of beats per minute.
cps A pointer filled by the function with the number of ticks per second.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.
