HP_ReadMIDIPort

To read all attributes of a MIDIPort event, identified by its index "id" in the eventlist.

HP_MIDIFILE_API UINT HP_ReadMIDIPort
(
HP_MIDIFILE_MODULE *hphandle,
int id,
int *time,
int *track,
int *port
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
id The index of the eventlist of the MIDIPort event.
time A pointer filled by the function with the time of the event.
track A pointer filled by the function with the track number.
port A pointer filled by the function with the port number.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.
