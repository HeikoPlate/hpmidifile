[all functions]

HP_ShiftSelected

To increase or decrease the time-position of all MIDI-event selected.

HP_MIDIFILE_API UINT HP_ShiftSelected
(
HP_MIDIFILE_MODULE *hphandle,
int ticks
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
ticks Number of ticks to be shifted.
ticks < 0: Decrease the time-stamp.
ticks > 0: Increase the time-stamp.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

It is not allowed to shift events to time-stamps < 0.
