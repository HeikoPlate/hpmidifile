HP_Free

To finish a MIDI-file session. The function destroys the data in the memory.

HP_MIDIFILE_API UINT HP_Free
(
HP_MIDIFILE_MODULE *hphandle,
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

It is not possible, to finish a MIDI-file session, if a play of MIDI events is still running.
