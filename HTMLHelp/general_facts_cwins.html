Cakewalk Instrument Definition Files: General Facts



Instrument Definition Files for the Sequencer Software Cakewalk/Sonar http://www.cakewalk.com are well structured text-files to get the names of voice-banks, patches, notes, drum-banks, drums, controller-events, RPN and NRPN for MIDI-Synthesizer and Keyboards. There are a lot of instrument definitions available in the internet.

HP MIDIFILE offers functions to get the texts of names by the numbers used in MIDI events. The first action is to load a Cakewalk Instrument Definition file (extension .ins) with HP_CWInsLoad(...). Then we are able with HP_GetDefs(...) to get a list of all MIDI-Instruments supported with the file. With HP_CWInsSetDef(...) we select an instrument now. After this the functions HP_CWInsGet...Name deliver the names of the objects.
