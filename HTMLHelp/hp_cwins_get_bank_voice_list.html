HP_CWInsGetBankVoiceList

To get a list of all voice-names which include a substring

HP_MIDIFILE_API UINT HP_CWInsGetBankVoiceList
(
HP_MIDIFILE_MODULE *hphandle,
char *search_string,
HP_CWBANKVOICE **bank_voice_list,
long *bank_voice_list_len
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
search_string The substring, which will be part of all voice-names, we want to get.
bank_voice_list A pointer to an array of HP_CWBANKVOICE structures, which is filled.
bank_voice_list_len A pointer to a long variable, which is filled with the number of elements of the bank_voice_list.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.

Remarks

The aim of this function is to implement a voice-browser as in Cakewalk. For the first call, the parameter bank_voice_list_len must be initialized by 0. The next call mostly is used to refine the search. Then the function has to be called with bank-voice_list and bank_voice_list_len, which is received with the last call. Look at the samples.

If the function succeeds, it allocates memory for the bank_voice_list, which must be destroyed by HP_Delete(..), if it is not needed for a refinement.
