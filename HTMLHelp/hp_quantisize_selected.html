HP_QuantisizeSelected

To quantisize selected note events.

HP_MIDIFILE_API UINT HP_QuantisizeSelected
(
HP_MIDIFILE_MODULE *hphandle,
int resolution,
bool start,
bool length
);

Parameters

hphandle Pointer to a HP_MIDIFILE handle.
resolution Specificies the resolution of quantisizing.
resolution = 1: 1/1 note.
resolution = 13: Triole with 1/1 note.
resolution = 2: 1/2 note.
resolution = 23: Triole with 1/2 note.
resolution = 4: 1/4 note.
resolution = 43: Triole with 1/4 note.
resolution = 8: 1/8 note.
resolution = 83: Triole with 1/8 note.
resolution = 16: 1/16 note.
resolution = 163: Triole with 1/16 note.
resolution = 32: 1/32 note.
resolution = 64: 1/64 note.
start start = true: Start-times are quantisized.
start = false: Start-times are not quantisized.
length length = true: Note lengths are quantisized.
length = false: Note lengths are not quantisized.

Return Value

Error code. On success the code HP_ERR_NONE.
See Error codes.
