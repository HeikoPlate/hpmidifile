========================================================================
    ppqn_master: A sample-utility to change the PPQN of MIDI-files 
                 using the functions of the package HP MIDIFILE
                 
		http://www.heikoplate.de/hpmidifile                 
========================================================================

The program is generated by Microsoft Visual C++ as MFC Windows application.

The HP_MIDIFILE library HP_midifile.lib has to be linked with your VC++ project. 

The header-file HP_midifile.h is added to the VC++ project. 
The dynamic link library HP_midifile.dll is copied into the directory of 
the executable file create_song.exe.

The line #include "HP_midifile.h" is inserted in ppqn_master.h
